# Scripts for Foskett Dace Population Genetics

1. gadma_3pop.sh : For running 3-population demographic model using GADMA
2. gadma_CI.sh : To get confidence intervals (CI) for parameters estimates for the most likely demographic model using bootstrap SFS
3. tassel.sh : To create neighbor joining (NJ) trees based on pairwise genetic distances among all Specked Dace individuals
4. vcfTools.sh : To filter sites and individuals from the VCF file using vcfTools
5. d3_stat.R : R-script to calculate D3 distances among trios of populations
